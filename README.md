# Forma-o_Power_BI_Analyst
Análise de Dados de BI aplicada com Power BI. A principal habilidade do profissional de dados está relacionada com a capacidade de transmitir os seus resultados de forma assertiva e clara. O objetivo é executar todo processo de ETL com a ferramenta de BI mais usada no mercado, apresentar relatórios e dashboards eficientes, organizados e elegantes.
